# ZhouLiRui 的导航站

欢迎来到 ZhouLiRui 的个人导航站集合，这里汇聚了多个实用且基于现代技术搭建的网站，为大家提供便捷的服务和体验。

## 站点列表

### 1. [pan.srczlrzh.xyz](https://pan.srczlrzh.xyz)  
- 平台：基于 Cloudreve 搭建  
- 存储支持：OneDrive  
- 加速服务：CloudFlare CDN  
- 备注：目前因 OneDrive E5 账户限制存在部分功能可能失效

### 2. [ai.srczlrzh.xyz](https://ai.srczlrzh.xyz)  
- 平台：基于 OpenWebUI 搭建  
- 部署方式：Docker 容器  
- 服务内容：AI 对话网站，支持多种智能交互体验  
- 加速服务：CloudFlare CDN

### 3. [zlrchat.zone.id](https://zlrchat.zone.id)  
- 平台：基于 VoceChat 搭建  
- 运行环境：Leaflow 云服务  
- 服务内容：即时聊天网站  
- 加速服务：腾讯云 CDN

### 4. [www.sla-api.zone.id](https://www.sla-api.zone.id)  
- 平台：基于 New API 搭建  
- 服务内容：AI 大模型 API 公益站，免费开放接口  
- 加速服务：腾讯云 CDN

### 5. [tool.zhoulirui.ggff.net](https://tool.zhoulirui.ggff.net)  
- 平台：基于 Edge One Page 搭建  
- 功能介绍：根据访问者 IP 显示所在国家及城市定位  
- 加速服务：腾讯云 CDN

---

### 备注  
- **zone.id** 和 **zhoulirui.ggff.net** 域名均由腾讯云 CDN 加速，确保访问快速稳定。  
- **srczlrzh.xyz** 域名则由 CloudFlare CDN 提供加速服务，保障全球网络访问顺畅。
